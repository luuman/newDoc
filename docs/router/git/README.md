# Git

## Git 分支命名

| Git      | 描述                                      |
| -------- | ----------------------------------------- |
| feat     | 新功能                                    |
| fix      | 修复                                      |
| docs     | 文档变更                                  |
| style    | 代码格式（不影响代码运行的变动）          |
| refactor | 重构（既不是增加 feature） 也不是修复 bug |
| pref     | 性能优化                                  |
| test     | 增加测试                                  |
| chore    | 构建过程或辅助工具的变动                  |
| revert   | 回退                                      |
| build    | 打包                                      |
| merge    | 合并分支                                  |
